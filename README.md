# Projeto Pessoal de Victor Aires:

## Descrição:
Extensão do Google Chrome para abrir todos os links que sempre utilizo com um click.

## Requisitos
- Listar os links desejados;
- Clicar em um botão e abrir todos na mesma janela do browser mas em abas separadas;
- Botão para adicionar um novo link na lista;
    - Digitando o link no input
    - Caso o input esteja vazio, adicionar o link da aba atual do chrome
- Botão para deletar um link da lista;
- Botão para deletar todos os links da lista;
